select a two-sided protocol for the send operation

This file define the msg ops functions. It is consisted of the following sections: send functions, receive functions and ops structure Send function

Parameters
[in]rxr_ependpoint
[in]tx_entrycontains information of the send operation
[in]use_p2pwhether p2p can be used
Returns
the RTM packet type of the two-sided protocol. Four types of protocol can be used: eager, medium, longcts, longread. Each protocol has tagged/non-tagged version. Some protocols has a DC version.
