const char * efa_strerror ( enum efa_errno  err)

Convert EFA error into a printable string.

Given a non-negative EFA-specific error code, this function returns a pointer to a static string that corresponds to it. This text is suitable for printing in debug/warning/error messages.

Parameters
[in]errAn EFA-specific error code
Returns
Pointer to a string corresponding to err
