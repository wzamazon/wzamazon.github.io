Convert EFA error into a printable string.

Given a non-negative EFA-specific error code, this function returns a pointer to a static string that corresponds to it. This text is suitable for printing in debug/warning/error messages. If an optional enriched error message string is provided, it will override the default message and be returned instead.

Parameters
[in]errAn EFA-specific error code
[in]enriched_errAn enriched error message. It should be obtained from a prior fi_cq_readerr call.
Returns
An error message with helpful debugging information specific to the error code
