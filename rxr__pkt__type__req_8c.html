calculate the exact header size for a given REQ pkt_entry

The difference between this function and rxr_pkt_req_hdr_size() is the handling of the size of req opt raw address header.

rxr_pkt_req_hdr_size() always use RXR_REQ_OPT_RAW_ADDR_HDR_SIZE, while this function pull raw address from pkt_entry's size.
