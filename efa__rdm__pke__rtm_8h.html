void efa_rdm_pke_rtm_update_rxe ( struct efa_rdm_pke *  pke,
struct efa_rdm_ope *  rxe 
)

Update RX entry with the information in RTM packet entry.

The following field of RX entry is updated: address: this is necessary because original address in rxe can be FI_ADDR_UNSPEC cq_entry.data: for FI_REMOTE_CQ_DATA msg_id: message id total_len: application might provide a buffer that is larger then incoming message size. tag: sender's tag can be different from receiver's tag becuase match only requires (sender_tag | ignore) == (receiver_tag | ignore) This function is applied to both expected and unexpected RX entry

Parameters
[in]pkeRTM packet entry
[in,out]rxeRX entry to be updated
