Detailed Description

a hashmap between sender address + msg_id to RX entry

This hash map is used on the receiver side to implement medium and runting protocols. Such protocol will send multiple RTM packets at the same time. The first RTM will be matched with an RX entry and will be inserted to this map, the later arriving RTM packet will use this hashmap to find the RX entry.
